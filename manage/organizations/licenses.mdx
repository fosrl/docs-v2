The Licenses page is where self-hosted Enterprise Edition users generate and manage license keys for their self-hosted Pangolin deployments.

## What This Page Is For

- Generate new license keys for self-hosted instances
- View existing license keys and instance mappings
- Check validity, type, limits, and expiration
- Refresh and search license records

## List View

The page includes:

- Search
- **Refresh**
- **Generate License Key**
- License table columns:
  - `License Key`
  - `Instance Name`
  - `Valid`
  - `Type`
  - `Users`
  - `Sites`
  - `Valid Until`

## How Teams Typically Use It

1. Generate a license key in Cloud.
2. Apply that key to the target self-hosted Enterprise instance.
3. Return to this page to confirm status, limits, and expiration.
4. Generate additional keys as needed for other instances/environments.

## Scope Clarification

- **Cloud** hosts the license management UI.
- **Self-hosted Enterprise Edition** consumes those generated license keys.
