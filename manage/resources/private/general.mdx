Private resources are reachable only by connected Pangolin clients in your organization.

## Create Private Resource Flow

The create dialog starts with:

- **Name**
- **Site**

Then you configure three tabs:

- **Network Settings**
- **Access Policy**
- **SSH Access**

## How to Use This Section

1. Choose the resource `Name` and `Site`.
2. Configure destination and port behavior in [Network Settings](/manage/resources/private/network-settings).
3. Assign who can access it in [Access Policy](/manage/resources/private/access-policy).
4. Configure SSH-specific behavior in [SSH Access](/manage/resources/private/ssh-access) if needed.
